Reproductive Health Application

By: Sylvia Martin

About the Project

The project is a mobile application for sexual and reproductive health that serves as a digital platform on which users can log key reproductive health data, view personalised statistics and insights, access built-in educational resources, and set reminders for health checkups and medication intake

The app's aim is to encourage users to take a more proactive, informed, and organised approach to managing their reproductive health.

Key Features

  • User authentication with email and password
  • Reproductive health tracking
  • Lifestyle tracking (sleep, water, diet, fitness)
  • Reminders and notifications
  • Educational articles and supplement information
  • Personalised statistics and insights

Technologies

  • Node.js
  • Express
  • MongoDB
